## About

Recursion is a programming technique that helps programmers understand how to decompose a problem into smaller parts and how to work backward from a solution. Recursive thinking builds a strong foundation for understanding and conceptualizing a lot of powerful algorithms. In this module, let us understand what recursion is and how we can find the best scenarios to use it in our algorithms.
